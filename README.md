# Online Retail Sales Analysis

## Project Overview
This project analyzes an online retail dataset to understand sales performance, product demand, and revenue trends.  
The analysis was performed using Excel and Power BI to generate insights and build an interactive dashboard.

## Tools Used
- Excel (Data cleaning, Pivot Tables)
- Power BI (Dashboard & Data Visualization)

## Dataset
The dataset contains information about online retail transactions including:
- Invoice number
- Product description
- Quantity
- Unit price
- Customer ID
- Country
- Invoice date

## Key Analysis
- Top 10 products by revenue
- Revenue by country
- Monthly revenue trend
- Sales vs Returns analysis

## Insights
- Certain products contribute significantly to overall revenue.
- The United Kingdom generates the highest revenue among all countries.
- Monthly revenue trends help identify peak sales periods.
- Returns have a noticeable impact on overall revenue.

## Dashboard
The Power BI dashboard provides interactive visualizations to explore sales performance across products, countries, and time.
